# 竞品 Review：China Highlights (chinahighlights.com)

> 抓取时间：2026-05-19  
> 来源：浏览器 Playwright 实抓首页 snapshot + 全页截图（D:/projects/tourism-landing/.playwright-mcp/）  
> 目的：作为我们 Tourism Landing v3.5 视觉与产品的对比参照，识别我们必须超越他的关键点。

---

## 1. 竞品产品定位

- 品牌：**China Highlights**（Highlights Travel 集团旗下，1998 年成立）
- 一句话定位：海外游客 inbound China 私家定制游 + 长行程团游报价站
- 商业模式：旅行社模式。强调“award-winning / best-rated / 10000+ reviews / 98.8% 5 星”
- 主转化漏斗：
  - 顶部 + 末尾大按钮 “Start Your Journey” → `/forms/create-my-trip.htm` 表单
  - Newsletter 订阅
  - 站内重型 SEO 内容引流（旅行指南 / 月份指南 / 城市页 / 主题页）
- 多语言：英 / 德 / 法 / 西 / 意 / 俄 / 日，子站独立部署

---

## 2. 它做对的地方（我们不要鄙视，要吸收）

1. **目的地与主题双轴导航完整**  
   顶部 menu 同时给“China Tours / Themes / Inspirations / About Us”，按目的地、主题、月份、行程长度、家庭/亲子/教育等多入口切。任何一个进站游客都能找到“给我看的”入口。

2. **行程长度切片做得很硬**  
   `5-day / 8-day / 10-day / 2-week / 3-week / 4-week / Visa-Free` 直接列在 Inspirations，回应的是真实用户问句“我有 X 天能怎么玩”。

3. **签证利好做了首屏第二条 banner**  
   “China Extends Visa-Free Policy Until the End of 2026”，列出可免签国家，直接消除“能不能来”这个最大顾虑。这条对海外游客非常致命。

4. **服务承诺打成了 4 人服务包**  
   “Personal Travel Designer / Local Storyteller / Private Driver & Car / 24/7 Travel Support”，把服务变成可记忆的 4 人小团队，比我们写一堆 trust badges 强。

5. **客户故事 + 评分背书路径长**  
   World Travel Awards、TripAdvisor Traveller’s Choice 多年奖项；Customer Stories 板块。海外游客对“是不是骗子”敏感，他们用奖项和故事压住。

6. **多语言子站独立**  
   不是一刀切翻译，是每个语种独立 SEO，针对各国游客生态。

---

## 3. 它做得不好的地方（我们的超越窗口）

### 3.1 视觉与设计

- **信息密度严重过载**：首页一屏塞顶部 mega menu 几十个链接，再叠 4 张大行程卡 + 8 个目的地卡 + 3 个 plan 引导 + 信任 logo + 服务承诺，没有视觉呼吸。
- **没有真正的视觉母题**：hero 是一张通用大图 + “China Highlights, Discovery Your Way!” 通用标题，没有当下季节 / 当下目的地的氛围感，更像是十年前 SEO 旅游门户。
- **风格不统一**：一些行程卡是真实摄影、一些是 Photoshop 拼合，配色从绿到红到金，缺少品牌视觉语言。
- **手机端体验粗糙**：mega menu 折叠后变成长长的列表流，移动端难浏览。
- **CTA 视觉太弱**：Start Your Journey 是普通 pill 按钮，没有“立即被服务”的引导感。

### 3.2 信息架构

- **入口太多反而模糊**：顶 menu 同时挤 China Tours / Themes / Inspirations / About，子菜单深 3 层，新游客很难快速决策。
- **首屏没有诊断式入口**：没有“我还没想好去哪 / 我只有 10 天 / 我带孩子 / 我喜欢自然”这种快速分流。游客必须自己读完再选，跳出率自然高。
- **没有“免费咨询”这条软入口**：核心 CTA 直接是“Start Your Journey 填表单”，跳过了高意向但还在看的人。这部分人会沉默离开。
- **价格完全不透明**：必须填表才能拿报价，对“先看看大概多少钱”的人筛走太多。
- **WhatsApp / 微信等亚洲常用通道不显眼**：海外华人和东南亚游客其实更愿意走 WhatsApp，他们却把电话和邮件做主。

### 3.3 详情页与跳转

- **行程详情页是长页 SEO 模板**：典型“概览 + 每日行程 + 报价 + 客户评价 + 行程地图”长滚动，没有交互式日历、没有可视化路线、没有按需替换。
- **行程之间不可对比**：4 条 14-day / 21-day 行程并排，但缺少“核心差异点对比表”，用户得自己点开对照。
- **没有“替我设计”的智能引导**：所有定制路径都通过同一个表单，不区分用户来源（亚洲 vs 北美 vs 老客户回购）。
- **客户故事和行程之间断链**：Customer Stories 在导航最下，没有与行程互相打通（哪条行程出过这种故事）。

### 3.4 转化与信任

- **信任元素堆叠但缺少“具体的人”感**：4-on-1 服务团队是抽象介绍，没有具体顾问的脸、名字、专长。
- **没有真实顾问 1v1 引导**：海外游客最怕“被群发表单”。我们如果能把“某位真实顾问的 WhatsApp 名片”推到首屏，会形成代差。
- **缺少“当季最佳推荐”动态层**：他们有 Seasonal Picks 入口但不在首屏首位置，季节差异化机会被浪费。
- **没有 itinerary preview 即时感**：没有“给我 3 选 1 的草稿行程”这种轻交互入口。

---

## 4. 我们必须超越他的 7 个维度

1. **首屏视觉冲击力 + 真实摄影优先（vs 他们的密集信息流）**  
   单张大幅真实景区摄影 + 留白，配 cinematic editorial 排版。第一眼传达情绪，不是堆功能。

2. **诊断式进入路径（vs 他们的统一表单）**  
   “Tell us where you want to go”、“I only have 10 days”、“I’m bringing my kids”、“I want nature, not cities” 等几个软分流入口。让没想好的人也能进。

3. **轻量优先 CTA 双轨（vs 他们的单点 Start Your Journey）**  
   - 主：免费咨询 / 定制行程
   - 副：WhatsApp 直聊 / 微信顾问
   - 两条都低门槛，且在首屏可见，不强制填长表单。

4. **真实本地顾问 + 具体面孔（vs 他们的抽象 4-on-1）**  
   首页就放 1–2 位本地顾问的姓名 / 语种 / 擅长目的地，下挂 “talk to advisor”，让信任有个人。

5. **签证 + 季节 + 行程长度三轴智能引导（vs 他们的 menu 硬目录）**  
   首屏内嵌“你来自哪个国家 → 我能免签几天 → 我建议你这条 X 天行程”的微互动，把他们 menu 里冗长的 5-day / 8-day / 10-day 列表换成动态推荐。

6. **行程对比与可视化路线（vs 他们的 SEO 长页）**  
   行程详情页要：互动地图 + 每日时间轴 + 替换段（“把 Xi'an 换成 Pingyao”）+ 多行程对比表。

7. **真实摄影优先 + AI 视觉退场流程（vs 他们的素材风格混乱）**  
   每张 hero / destination / advisor 图片都进 attribution，不允许 stock 笑脸，且要符合统一光影方向。AI 图只在原型阶段用，参考 `docs/visual-references/REAL-IMAGE-SOURCING.md`。

---

## 5. 不要犯他们犯过的错（明确禁止项）

- 首屏不要 mega menu 硬塞 50 个链接。
- 首屏不要 8 张同等级目的地卡片堆。
- 首屏不要单一 “Start Your Journey” 表单作为唯一入口。
- 详情页不要做成 SEO 长滚动模板，要交互化。
- 不要把价格完全隐藏到表单后。
- 不要靠抽象 4-on-1 服务承诺替代“具体顾问 + WhatsApp”。
- 不要让客户故事与行程互相孤立。
- 不要奖项 logo 大于真实摄影主视觉。

---

## 6. 落到我们 Tourism Landing 的下一步

短期（首屏原型 v3.5 → Figma Make）：

1. 用 `mood-only/` 8 张图作为视觉方向锚，喂给 Figma Make。
2. 首屏出 1–2 个版本，分别测：
   - A：地图居中 + 探索导向（沿着 v3 brief）
   - B：单张大幅真实摄影 + concierge 便签 + WhatsApp 卡（放弃地图主视觉，对标他们 hero 但摄影质感强一档）
3. 客户 review 选定方向。

中期（详情 + 行程页）：

1. 行程详情页拆 3 个组件：交互式日历 / 可视路线地图 / 段替换 (swap segment)。
2. 多行程对比表（不少于 3 行程并排）。
3. 顾问名片组件，可全站复用。

长期（转化与运营）：

1. 签证 + 季节 + 国别智能引导。
2. WhatsApp / 微信即时入口接入。
3. 真实顾问 + 真实客户故事打通行程页。
4. 真实摄影替换 mood-only AI 图，按 `REAL-IMAGE-SOURCING.md` 走。
